======================
HOW TO BUILD MICEAMAZE
======================

You need:
- GCC C++ compiler and GNU Make
- OpenGL
- SDL
- SDL_mixer
- SOIL
- QuesoGLC
- DejaVu font installed
- rsvg
-ImageMagick

Without installing:
make
./miceamaze

With installation (requires root privileges):
make
sudo make install
miceamaze
